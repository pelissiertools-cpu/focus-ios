# Build Project

Build the Focus iOS app for simulator using XcodeBuildMCP.

## Instructions

1. Use `mcp__xcodebuildmcp__discover_projects` to find the Xcode project
2. Use the appropriate build tool for iOS Simulator
3. Report any build errors with file paths and line numbers
4. If errors occur, suggest fixes based on the error messages

## Expected Output

- Build success confirmation
- Or detailed error report with actionable fixes

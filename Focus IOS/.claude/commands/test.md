# Run Tests

Run all tests for the Focus iOS app using XcodeBuildMCP.

## Instructions

1. Use `mcp__xcodebuildmcp__discover_projects` to find the Xcode project
2. Use the test tool to run unit tests
3. Report test results: passed, failed, and skipped counts
4. For failures, provide the test name, file location, and failure reason

## Expected Output

- Test summary with pass/fail counts
- Detailed failure information if any tests fail
